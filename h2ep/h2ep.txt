*** H2 Event Protokoll ***

Basis:
	Das h2ep basiert auf einer TCP-Verbindung.
	(Neu: Es wird auch basierend auf WebSocket-Protokollen verwendet.)

Konzept, Inhalt:
	Das Konzept von h2ep ist, dass einzelne Events zwischen Server und Client ausgetauscht werden.
	Ein Event besteht aus einer Eventbezeichnung und Daten in Form eines JSON-Strings.

Syntax:
	Ein Event beginnt mit einer eingeklammerten Zahl mit der Länge der Eventbezeichnung, welche gleich darauf folgt.
	Nach der Eventbezeichnung folgt die Länge der Daten, welche gleich im Anschluss stehen.
	
		"(" + Bezeichnungslänge + ")" + Bezeichnung + "(" + Datenlänge ")" + JSON
		
	Beispiel
		
		(7)Startup(66){"Zeit":"Sa 21.04.2018 15:42","Info":"Irgendwas wurde gestartet."}
